# OOP experiences
FIT2099 is the unit that introduce to object-oriented design principles and their application to the construction of quality software. I have learnt the fundamental concepts that underlie modern object-oriented languages, how to implement designs using at least one, and how to use standard notation to illustrate my designs[1].

[1] https://handbook.monash.edu/2026/units/FIT2099?year=2026

# About the product
- My project is inspired by the classic fantasy game Rogue, which created the 'rogue-like' genre of text-based adventure games that were popular before modern computer graphics.
- The game engine was designed by the teaching team, I was mainly created the characters, items, weapons, and other additional events.
- To find more details and documents, check out the docs/design and docs/javadoc-A3.
