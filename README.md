# OOP experiences
FIT2099 is the unit that introduce to object-oriented design principles and their application to the construction of quality software. I have learnt the fundamental concepts that underlie modern object-oriented languages, how to implement designs using at least one, and how to use standard notation to illustrate my designs[1].

[1] https://handbook.monash.edu/2026/units/FIT2099?year=2026

What I have achieved:
- Designed small-to-medium software systems iteratively using object-oriented principles, mapping architectures with standard UML class and interaction diagrams.
- Evaluated design quality based on user requirements and the effective application of object-oriented design concepts.
- Implemented robust object-oriented designs in Java, leveraging core constructs including abstraction, information hiding, inheritance, and polymorphism.
- Systematically refactored and debugged codebases using language tools to optimize performance and maintainability.
- Collaborated in peer environments using modern software engineering tools, including IDEs, UML modeling software, and version control systems (Git).
